## Installation and Setup Instruction
#### Run the following commands in your terminal

`git clone https://github.com/Gaursafal/butane.git`

`cd VentureBeat Clone`

`npm install`

`npm start`

## Libraries installed

- Material UI
- Axios
- React router dom
- Styled components
